## Descripción

Aplicación web de tipo **Gestor de Tareas** desarrollada como entrega del examen de la semana 9. Permite al usuario administrar sus tareas con prioridades, estados y persistencia local.

## Funcionalidades

- ✅ **Agregar tareas** con título, descripción y prioridad (alta / media / baja)
- ✅ **Marcar como completadas** mediante checkbox
- ✅ **Eliminar tareas** individualmente
- ✅ **Filtrar** por estado: Todas, Pendientes, Completadas
- ✅ **Persistencia** automática en `localStorage` (los datos se conservan al recargar la página)
- ✅ **Diseño responsive** adaptado a dispositivos móviles

## Tecnologías

| Tecnología | Uso |
|-----------|-----|
| HTML5 | Estructura semántica |
| CSS3 | Estilos y diseño responsive |
| JavaScript (ES6+) | Lógica de la aplicación |
| localStorage | Persistencia de datos |
